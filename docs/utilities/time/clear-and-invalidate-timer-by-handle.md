# ClearAndInvalidateTimerByHandle

清除设置的定时器。

操作对象是 Kismet System Library

![](./images/clear-and-invalidate-timer-by-handle.png =265x)

## 输入
| 引脚 | 参数 | 类型 | 描述 |
| -- | -- | -- | -- |
| <IconExec /> | In | Exec | 
| <IconPin color="#0057c5" /> | Handle | Timer Handle Structure (by ref) | 需要清除的定时器的引用。 |

## 输出
| 引脚 | 参数 | 类型 | 描述 |
| -- | -- | -- | -- |
| <IconExec /> | Out | Exec |
