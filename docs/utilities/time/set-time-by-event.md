# Set Time by Event

设置一个执行委托的定时器。设置已存在的定时器时，会使用最新的参数覆盖原有定时器参数

操作对象是 Kismet System Library

![Set Time by Event](./images/set-timer-by-event.png =365x)

## 输入
| 引脚 | 参数 | 类型 | 描述 |
| -- | -- | -- | -- |
| <IconExec /> | In | Exec | 
| <IconDelegate /> | Event | Delegate | 需要执行的委托 |
| <IconPin color="" /> |  |  | 

## 输出
| 引脚 | 参数 | 类型 | 描述 |
| -- | -- | -- | -- |
| <IconPin color="" /> |  |  |
