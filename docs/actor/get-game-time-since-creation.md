# Get Game Time Since Creation

## 输入
| 引脚 | 参数 | 类型 | 描述 |
| -- | -- | -- | -- |
| <IconPin color="#00a8f4" /> | Target | Actor Object Reference | Actor 对象引用 |

## 输出
| 引脚 | 参数 | 类型 | 描述 |
| -- | -- | -- | -- |
| <IconPin color="#7ef48f" /> | Return Value | Float | Actor 在游戏中从创建到当前时间的秒数，相对于游戏时间 |
