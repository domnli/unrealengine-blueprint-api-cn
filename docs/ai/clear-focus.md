# ClearFocus

Clears Focus, will also clear FocalPoint as a result

操作对象是 AIController 

![](./images/clear-focus.png =365x)

## 输入
| 引脚 | 参数 | 类型 | 描述 |
| -- | -- | -- | -- |
| <IconExec /> | In | Exec |
| <IconPin color="#00a8f4" /> | Target | AIController Object Reference | AIController 引用 |

## 输出
| 引脚 | 参数 | 类型 | 描述 |
| -- | -- | -- | -- |
| <IconExec /> | Out | Exec |
